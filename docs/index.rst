sphinxmixcrypto
===============

This is a crypto library for writing mix networks.
The code was forked from Ian Goldberg's reference implementation.


Getting sphinxmixcrypto:
------------------------

Code available at https://github.com/applied-mixnetworks/sphinxmixcrypto

- ``git clone https://github.com/applied-mixnetworks/sphinxmixcrypto.git``
- ``pip install sphinxmixcrypto``
